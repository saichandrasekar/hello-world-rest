# Hello World REST (hello-world-rest v1.0.0)

## Tech Stack
- Java 17
- Spring Boot 3.0.0

## Onboarding

### Software Tools:

- Java JDK 17
- Apache Maven
- Docker

### Installation Steps

1. Build the code

```
$ mvn clean install
```

2. Build Docker Image

```
$ docker build .
```

### Deploy REST API

1. Run as docker container

```
$ docker run 
```
